Increased Efficiency for Bread Baking Process

Case Number 278

last update: February 23, 2009

Easy switching of temperature settings by bread type

Temperature settings can be registered for up to eight banks by using the bank function of the E5EN-H Temperature Controller. Easily perform external temperature settings according to the type of bread.

Know-How

The temperature settings are easily switched according to the type of bread by using the bank function of the E5EN-H Temperature Controller.

Benefit / Efficiency

The user can easily switch the baking temperature for each type of bread.
