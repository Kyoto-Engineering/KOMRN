Redundancy in Temperature Control and Temperature Monitoring

Case Number 281

last update: April 13, 2015

Low-cost safety measures using temperature control redundancy in the frying process

Low-cost redundant temperature control monitoring for fryers is performed by using two Temperature Controllers.

Know-How

Oil temperature is controlled by using an Temperature Controller, and notification is provided by using a buzzer with alarm output if a constant temperature is exceeded. And, with an E5C2 Temperature Controller, a cutoff valve is closed and the gas burner is stopped if the temperature exceeds 200℃.

Benefit / Efficiency

Using an E5C2 for temperature monitoring to prevent overheating enables low-cost temperature management redundancy.
