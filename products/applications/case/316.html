Monitoring Ship Engine Exhaust Temperatures

Case Number 316

last update: July 8, 2009

This application monitors the exhaust temperature at a ship engine exhaust pipe.

A Temperature Sensor detects the temperature at the engine exhaust pipe. A Temperature Indicator displays the values. The system outputs an alarm if there is a temperature abnormality.

Know-How

The system uses two selectable display colors (green and red) and a Temperature Indicator with a position meter that makes it easy to intuitively see the status using a bar chart.

Benefit / Efficiency

You can monitor the temperature at ship engine exhaust pipes.
