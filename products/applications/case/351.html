Coin-operated Car Wash

Case Number 351

last update: October 7, 2010

Bit Logic and Timer Application

The ZEN can be used to change the operating time depending on the number of coins inserted. If there are unexpected power interruptions, the remaining time will not be reset.

Know-How

When you use a holding timer (#) with holding bits (H) in self-holding programming, the remaining time will not be reset even if there are unexpected power interruptions.

Benefit / Efficiency

Because wiring is not necessary individually for the self-maintaining bit, you can reduce the wiring time greatly.
