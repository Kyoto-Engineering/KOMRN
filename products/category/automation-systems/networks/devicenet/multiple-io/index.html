MULTIPLE I/O

MULTIPLE I/O Terminals can be built from a DeviceNet Communications Unit and building-block Slaves. I/O Units can be easily added to or removed, enabling the simplest maintenance possible.

MULTIPLE I/O List

There are 6 products of MULTIPLE I/O.

Communications Unit DRT1-COM

Connects to a Total Maximum of Eight Digital I/O, Analog I/O, and Relay Output Units Compatible with MULTIPLE I/O TERMINAL.

Digital I/O Units GT1-ID / OD

Digital I/O Units Compatible with MULTIPLE I/O TERMINAL

Analog I/O Units GT1-AD / DA

Analog Input/Output Units Compatible with MULTIPLE I/O TERMINAL

Temperature Input Units GT1-TS04[]

Temperature Input Units for use with MULTIPLE I/O TERMINAL

Counter Unit GT1-CT01

A Counter Unit Supporting Encoder Input for Use with MULTIPLE I/O TERMINAL

Relay Output Units GT1-ROS16 / ROP08 / FOP08

Relay Output Unit Compatible with MULTIPLE I/O TERMINAL
