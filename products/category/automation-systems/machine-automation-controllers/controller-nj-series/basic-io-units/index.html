Basic I/O Units

Basic I/O Units are available for essentially any application and wiring method.

Basic I/O Units List

There are 6 products of Basic I/O Units.

CJ-series Input Units CJ1W-ID / IA

A wide range of CJ series input units are available featuring high-speed input and suitability for various applications. This unit is available for CJ/NJ controllers.

CJ-series Output Units CJ1W-OC / OA / OD

A Wide Range of Basic Output Units for High Speed Output and Different Applications. This unit is available for CJ/NJ controller.

CJ-series Mixed I/O Units CJ1W-MD

A Wide Range of Basic Mixed I/O Units for Different Applications and Wiring Methods. This unit is available for CJ/NJ controller.

CJ-series Interrupt Input Units CJ1W-INT01

High-speed Starting of I/O Interrupt Tasks. Use Up to Two Units for a Total of Up to 32 Interrupt Inputs. This unit is available for CJ/NJ controller.

CJ-series Quick-response Input Units CJ1W-IDP01

Capture Input Signals Shorter Than the Cycle Time. This unit is available for CJ/NJ controller.

CJ-series B7A Interface Unit CJ1W-B7A

It transmits 16 points of I/O signals per word with a pair of cables. Easy and minimized wiring effort. This unit is available for CJ/NJ controller.
