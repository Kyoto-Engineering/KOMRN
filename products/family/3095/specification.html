Recommended System Requirements

Item Recommendation
OS Windows XP SP3 or higher
Windows Vista SP2 or higher (32-bit edition only)
Windows 7 (32-bit edition only)
Microsoft .NET Framework 3.5 is required.
This makes it impossible to run the software on Windows 2000 or earlier versions of Windows.
Processor Intel Core 2 DUO, 2.0 GHz or better
Memory 1 GByte or more
Other items The software is provided on a DVD, so a device to read the DVD is required.
A LAN interface is required to automatically collect data.
A device to read SD cards is required to use SD cards to collect data.

This software supports five languages

Japanese, English, Chinese, Korean, and Taiwanese.
