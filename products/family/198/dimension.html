Mounting Height

Use the P2RF-[]-E or P2R-[]7P to mount the H3RN. When ordering any one of these sockets,replace "[]" with "05" for SPDT or "08" for DPST-NO.
The P2RF-[] cannot be used because the hook is a different shape.

P2RF-[]-E

Note: 1. The value shown indicates the dimension for the P2RF-05-E with the PFP-[]N Mounting Rail.
              The value is 71.5 mm when using the PFP-N[]2.
          2. The value shown in parentheses indicates the dimension for the P2RF-08-E with the PFP-[]N Mounting Rail.
               The value is 75.5 mm when using the PFP-N[]2.
