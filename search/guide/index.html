Search Guide

This is a guide to using the search function.

Using the Search Function

Enter key words, such as product features, model number, or name, in the text area and click the Search Button.
Information that matches the keywords will be displayed. When using multiple keywords, separate them with spaces. There are three search methods on the site described below.
When a new tab is selected, the keywords are inherited and matching information is displayed.

Site Search — Searching the entire of this site

Site Search finds information throughout the entire of this site.
The number of hits for the key words that were input are displayed along with a list of pages.

Product Search — Searching for Product Information on this site

Product Search finds product information that OMRON corporation provides on a global basis.

(1) Pages with the Product Search Function

Product Search can be used on the Products, Item Index, and Lineup pages.

(2) Matching Categories

If the key word input matches a category name on the site, the results will be displayed.

(3) Display of the Number of Hits and Product Information

The number of hits for products matching the keywords will be displayed along with a list of product information.

When the partial code matches an item code
  • The item is displayed.
  • A link to the item page is provided.
  • A link to the related catalogue page is provided
When the partial code matches a catalogue code
  • The catalogue is displayed.
  • A link to the catalogue page is provided.
  • A link to the related item index is provided.
[Note]
  • "Catalogue" : e.g., E2E, E3Z
  • "Item" : e.g., E2E-C1B1 2M, E3Z-D61 2M
(4) Regarding the suggest function

This function suggests catalogue and item codes to aid your search, and provides links to catalogue and item pages.

  • On the Products Search page
    Inputting a partial code will display a list of matching catalogue and item codes.
    Find all products on the OMRON Industrial Automation Global Site.
  • On the Item Index, Lineup page
    Inputting a partial item code will display a list of matching item codes.
    Only items related to the displayed catalogue are suggested as possible matches.

IA Global Search — Searching throughout Country Websites of OMRON Industrial Automation

IA Global Search finds information and documents throughout Country Websites of OMRON Industrial Automation.

(1) Search result

The number of hits for the key words that were input are displayed along with a list of pages.
[login-required] will appear in the list for membership contents which requires login.

(2) Refining Search

From ‘Select Country in’ pull down menu, it is possible to limit search results by region or countries.

[Note]
  • The displayed lists may not be properly displayed depending on environment and conditions.
