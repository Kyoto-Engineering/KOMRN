Download UPS Setting Utility

The UPS Setting Utility is a software for performing various UPS settings. You can use this software to easily change the UPS settings.

Example: Stop the UPS without using the Shut-down Software in case of power outage.
→ Set Maximum Backup Duration.

You can also backup the UPS settings to a file, or transfer the settings to another UPS.

[Utility Functions]

  1. Transmit commands
  2. Save settings to a backup file
  3. Recover settings from a backup file
  4. Retrieve UPS settings
  5. Return to UPS default settings
