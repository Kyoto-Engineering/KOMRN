Question

Can you connect the K2CU Heater Element Burnout Detector's control power supply from a circuit other than the heater circuit?

Answer

ï»¿

The control power supply for the K2CU is generally the same as the heater circuit power supply.

Explanation:

The K2CU Series consists of two types of Heater Element Burnout Detector: one without gate input terminals and one with gate input terminals, as described below.

1. Detectors with Gate Input Terminals (K2CU-F[][]A-[], K2CU-P)

Synchronization must e made with Solid State Relays and magnet contactors for heater control.
The same power supply must be therefore be used.

2. Detectors with Gate Input Terminals (K2CU-F[][]A-[]GS)

All models with gate input terminals are equipped with a voltage fluctuation correction function. Because this function corrects the standard value of the internal setting circuits, if it does not use the same power supply, errors will occur in the operating values due to fluctuations in the control power supply voltage. This is why the same power supply must be used.
