Question

What is a substitute product for H3BG Star-delta Timers?

Answer

﻿

1.It depends on H3BG model. Refer to the following table.

Note:

1.H3BG-8-30 and H3BG-8H-31 have special external connections, e.g., for power supplies. The external connections are the same as H3CR-G8L-30 Solid-state Timers and H3CR-G8EL-31.

2.The bottom row gives recommended substitute products or models with substitute functions. For details, refer to the following information.

2.The substitute product in column (1) is H3CR-G8L. For H3BG-8-30 with a special terminal arrangement, the substitute product is H3CR-G8L-30. This means that previously UL-certified timers will no longer satisfy the safety standard.
The terminal wiring will have to be changed, but it is recommended that H3BG-8-30 is replaced with H3CR-G8L with a standard terminal arrangement.

3.The substitute product in column (2) is H3CR-G8EL. For H3BG-8H-31 with a special terminal arrangement, the substitute product is H3CR-G8EL-31. This means that previously UL-certified timers will no longer satisfy the safety standard.
The terminal wiring will have to be changed, but it is recommended that H3BG-8H-31 is replaced with H3CR-G8EL with a standard terminal arrangement.

4.There is no substitute in column (3) because H3CR-G has no top screw terminals.
However, a substitute timer function can be achieved by using H3CR-G8L and P2CF-08 Front Connecting Sockets, and wiring to the Front Connecting Sockets.
If a top screw terminal is required, see whether H3DE-G Solid-state Timer can be used.

5.There is no substitute in column (4) because the H3CR-G has no top screw terminals.
However, a substitute timer function can be achieved by using H3CR-G8EL and P2CF-08 Front Connecting Sockets, and wiring to the Front Connecting Sockets.
Even if a top screw terminal is required, H3DE-G cannot be used because it doesn’t have instantaneous contacts.
