What is a substitute product for MM2T Delay Relays?

Note:The bottom row gives recommended substitute products or models with substitute functions. For details, see the following information.

2.There is no substitute product in column (1) because MM2T utilize the fluid characteristics of oil, which significantly differs in principle and construction with the current mainstream digital timers.
However, a substitute function can be created by using H3YN-2 Solid-state timers with relays such as MM2.
From the contact specifications, it is recommended that MM2 is used as the substitute product for MM2T, MM2B for MM2TB, and MM2P for MM2TP.
If the actual operating conditions satisfy H3YN-2 timers contact specifications, relays such as MM2 are not required. In this situation, H3CR-A8 Solid-state timers could be used. If 8PFA sockets are used as the sockets for MM2TP, Y92F-42 Adaptors can be used instead of H3CR-A8 Sockets, and no wiring will need to be changed.

3.There is no substitute product in column (2).
However, a substitute function can be created by using H3YN-2 with relays such as MM2X.
H3YN-2 must be used in ON-delay operation mode, and mounted using PYF08A Surface Mounting Sockets.
From the contact specifications, it is recommended that MM2X is used as the substitute product for MM2XT, MM2XB for MM2XTB, and MM2XP for MM2XTP.
