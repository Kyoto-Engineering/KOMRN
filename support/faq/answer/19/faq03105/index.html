Note:

1.The SPDT' time-limit contacts output after a time lag of 3% FS max. for SYS-3, SYS-C-3, and SYS-F-3.

2.SYS-T-P, SYS-C-T-P, SYS-T-P-2, and SYS-C-T-P-2 are connected by metal connectors.

3.SYS-T-P3 and SYS-C-T-P3 are connected by 8-pin molded connectors.

4.The bottom row gives recommended substitute products or models with substitute functions. For details, refer to the following information.

2.The substitute product in column (1) is H3AM-NS Analog Timers. The power supply is a 100 to 240 VAC, 50/60 Hz common supply. H3AM model come in three time ranges and can be selected from the A, B, or C series. The terminal arrangements for external connection are not compatible.

If SYS or SYS-3 uses a combination of time-limit contacts and instantaneous contacts that exceeds the output of a DPDT contacts, additional contacts must be externally installed for H3AM because the output of a DPDT is the maximum limit for H3AM.

If SYS-3 has time-lag output, a time-lag circuit must be externally installed for H3AM because H3AM does not have a time-lag output function.

If SYS-T-P uses a connector, external wires must be connected to the terminal of H3AM because H3AM does not use connectors.

If SYS-F is a surface mounting model, there is no recommended substitute product because there are no surface mounting H3AM models. See whether a flush mounting model can be used.

3.The substitute product in column (2) is H3AM-NSR with electric reset. The motor and clutch are separately connected. The power supply is a 100 to 240 VAC, 50/60 Hz common supply. H3AM comes in three time ranges and can be selected from the A, B, or C series. The terminal arrangements for external connection are not compatible.

If SYS model has the suffix "-2" such as SYS-2, an inverted signal must be created externally to the timer because H3AM-NSR reset input which equates to SYS clutch input is inverted.

If SYS-T-P-2 or another model with a connector was used, H3AM will require the terminals to be externally wired because they do not use connectors.

If SYS-F-2 or another surface mounting model was used, there is no recommended substitute product because there are no surface mounting H3AM models. See whether a flush mounting model can be used.

4.There is no substitute product in column (3) because there are no H3AM models with repetitive operation. However, by treating H5BR or H5CX digital output of 0.5 s as a one-shot output, SYS-RP self-reset model as mode B, and SYS-R-RP electric reset model as mode B-1, a repetitive operation can be replicated. See whether this is suitable for your application because the size and mounting method are completely different.

5.There is no substitute product in column (4) because there are no H3AM models with instantaneous operation or time-limit resetting. However, by setting the output mode of H3CR-A8 Solid-state Timer to mode E (i.e., interval operation), the instantaneous operation and time-limit resetting functions can be replicated. However the operation of the timer will no longer be backed up for power blackouts.
See whether this setup is suitable for your application because the size and mounting method are completely different.
