Note:The bottom row gives recommended alternative models or models with alternative functions. For details, see the following information.

2.The alternative model in column (1) is the H3CR-A. Because the length of the body is about 12 mm shorter than that of the H3BA, the Y92H-7 Hold-down Clips should be used instead of the Y92H-1 Hold-down Clips when using the PL11 Socket.

3.There is no alternative in column (2) because the H3CR-A has no top-wired terminal. However, an alternative timer function can be achieved by using the H3CR-A and the P2CF-11 Front Connecting Socket, and wiring to the Front Connecting Socket. If a top-wired terminal is required, see whether the H3DE-M/S Solid-state Timer or H2C-F Motor Timer can be used.

4.The alternative model in column (3) is the H3CR-A8. Because the length of the body is about 12 mm shorter than that of the H3BA-8H, the Y92H-8 Hold-down Clips should be used instead of the Y92H-2 Hold-down Clips when using the PF085A Socket and the Y92H-7 Hold-down Clips should be used instead of the Y92H-1 Hold-down Clips when using the PL08 Socket.

5.The alternative model in column (4) is the H3CR-A8E. The Hold-down Clips to be used are the same as those described for the H3BA-8 in column (3).

6.The alternative model in column (4) is the H3CR-A8E. The Hold-down Clips to be used are the same as those described for the H3BA-8 in column (3).

1.The H3CR-A has no model that is equivalent to the top-wired terminal model of the H3BA-FA.

The H3CR Series also has a long-body model, which is distinguished from the short-body model by adding the letter "L" to the model number.
