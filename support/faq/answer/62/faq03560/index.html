Question

What Controllers can be used with ES2-THB Temperature/Humidity Sensors?

Answer

ï»¿

Humidity

The humidity output of ES2-THB is a 1 to 5-V linear output for a humidity of 0% to 100%.
Select a Temperature Controller that supports a 1 to 5-V input, and set the input range to 1 to 5 V for the Controller.

Connectable Controllers

E5CN-[]L Series:

K3MA-J Series:

K3GN-series:

General-purpose 48 x 48-mm Temperature Controllers with PID control

Process Meters with 14.2-mm character height, high visibility, and comparative outputs or display only

Compact 48 x 24-mm DIN Digital Panel Indicators with comparative outputs

A 24-VDC power supply is required for the Sensor.

Temperature

The temperature output of ES2-THB is the output of a Pt100 three-wire platinum resistance thermometer. Select a Temperature Controller that supports input from a Pt100 three-wire platinum resistance thermometer.
Set the Controller input range to Pt100.
