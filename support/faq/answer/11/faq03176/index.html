Note:

1.Model numbers with "L" have a maximum counting speed of 30 Hz (cps). Other models are 1 kHz (cps). Only the models with a 20-Hz maximum counting speed have multi-voltage inputs.

2.The bottom row gives recommended substitute products or models with substitute functions. For details, refer to the following information.

2.The substitute product in column (1) is H7EC-N.

If a wire-wrap terminal was used, Y92S-37 Wire-wrap Terminals (two in one set) must be separately purchased and attached to the terminal block.
If manual reset is not used, the key-protect switch must be set to ON.
If the maximum counting speed was 1 kHz (cps), the counting speed switch must be set to 1 kHz (cps).

3.The substitute product in column (2) is H7EC-NV.

If a wire-wrap terminal was used, Y92S-37 Wire-wrap Terminals (two in one set) must be separately purchased and attached to the terminal block.
If manual reset is not used, the key-protect switch must be set to ON.
If the maximum counting speed was 1 kHz (cps), the counting speed switch must be set to 1 kHz (cps).

4.The substitute product in column (3) is H7EC-NFV.

If the high level was 6 to 24 VDC, H7EC-NV with DC voltage input must be used because if the DC high level of H7EC-FBV with multi-voltage input was 6 to 240 VDC, it becomes 24 to 240 VDC for H7EC-NFV.
If a wire-wrap terminal was used, Y92S-37 Wire-wrap Terminals (two in one set) must be separately purchased and attached to the terminal block.
If manual reset is not used, the key-protect switch must be set to ON.

5.There are two substitute products in column (4). The substitute product for H7EC-P is H7EC-NP, and H7EC-NLP for H7EC-LP because Totalizers with direct mounting on the PC board do not have the counter speed switch.
