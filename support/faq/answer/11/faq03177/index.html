Note:

1.Discontinued models with no number indicate that the time is displayed in hours. Models with the suffix "1" indicate that the time is displayed in hours, minutes, and seconds. The suffix "2" indicates hours and minutes, and the suffix "3" indicates days and hours.

2.The bottom row gives recommended substitute products or models with substitute functions. For details, see the following information.

2.If H7ET-B1 or H7ET-B2 was used, the substitute product is H7ET-N1 with time display units switchable between h/min/s and h/min. If other models were used, the substitute product is H7ET-N with time display units switchable between h and day/h. (See column (1).)
If a wire-wrap terminal was used, Y92S-37 Wire-wrap Terminals (two in one set) must be separately purchased and attached to the terminal block.
If manual reset is not used, the key-protect switch must be set to ON.
The time display unit must be set using the time-range switch.

3.If H7ET-BV1 or H7ET-BV2 was used, the substitute product is H7ET-NV1 with time display units switchable between h/min/s and h/min. If other models were used, the substitute product is the H7ET-NV with time display units switchable between h and day/h. (See column (2).)
If a wire-wrap terminal was used, Y92S-37 Wire-wrap Terminals (two in one set) must be separately purchased and attached to the terminal block.
If manual reset is not used, the key-protect switch must be set to ON.
The time display unit must be set using the time-range switch.

4.If the H7ET-FBV1 or H7ET-FBV2 was used, the substitute product is H7ET-NFV1 with time display units switchable between h/min/s and h/min. If other models were used, the substitute product is H7ET-NFV with time display units switchable between h and day/h. (See column (3).)
If the high level was 6 to 24 VDC, H7ET-NV or H7ET-NV1 with DC voltage input must be used because if the DC high level of H7ET-FBV with multi-voltage input was 6 to 240 VDC, it becomes 24 to 240 VDC for H7EC-NFV.
If manual reset is not used, the key-protect switch must be set to ON.
The time display unit must be set using the time-range switch.

5.The substitute product in column (4) is H7ET-NP.
