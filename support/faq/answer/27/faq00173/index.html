Question

What method is used to store the screens in the NT30, NT31, NT620, and NT631 Programmable Terminals?

Answer

ï»¿

Screen data is stored in flash memory, so there is no battery backup.

The following data is retained by the battery:

Numeral memory table and character string memory table data in the Programmable Terminal

Display history and alarm history data

Calendar/clock settings

The Battery life expectancy is about 5 years at 25°C.
A C500-BAT08 Lithium Battery is used to back up this data.

The service life will be shorter when the Programmable Terminal is used at higher temperatures.

Replace the Battery as required based on the conditions in which the Programmable Terminal is being used.
