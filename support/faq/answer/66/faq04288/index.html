Compatible Combinations of CPU Units and Expansion I/O Units

The Expansion I/O Units that can be connected to V2 CPU Units are different from those that can be connected to V1 CPU Units and previous CPU Units. Expansion I/O Units with AC Inputs cannot be connected to V2 CPU Units with DC Power Supplies.
Expansion I/O Units cannot be connected to Economy-type CPU Units.
