Question

Is it possible to connect two CPM1A Programmable Controller Units with a RS-232C cross cable through an adapter, process an input to one CPM1A Unit, and output the result from the other CPM1A Unit?

Answer

ï»¿

Data in the LR Area can be transferred using a CPM1A 1:1 link. Transfer the necessary data to the LR Area to use this function. DM 6650 must be set to use a 1:1 link. In this case, the baud rate will be set to 19,200 bps and cannot be changed.
