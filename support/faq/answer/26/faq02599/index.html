Question

How can I connect a C200H Programmable Controller's CPU Unit with an IBM PC/AT or compatible computer?

Answer

ï»¿

1.Mount the C200H-IP007 Peripheral Interface Unit to the C200H and connect the CQM1-CIF02 Connecting Cables.

2.Mount the C120-LK201-V1 Host Link Unit to the C200H and connect the XW2Z-200P-V Connecting Cables.

The C200H CPU Unit and the computer cannot be connected using only a cable. An interface is always necessary. In either case, the Host Link communications mode is used.
