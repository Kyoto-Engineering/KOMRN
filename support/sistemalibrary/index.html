SISTEMA Library

What is reliability data for safety of machinery?

In order to meet the requirements for CE marking and making a declaration of conformity to EU Machine Directive, it is indispensable to validate the Safety Performance Level (PL) for Safety Related Parts of Control System (SRP/CS) of the machine. PL is derivable from reliability data for safety of control components which are used in the SRP/CS, in the context of the normative EN/ISO 13849-1: 2008.

What is SISTEMA?

SISTEMA is a software tool to calculate PL. It is provided by IFA as free software and considered as the most widely recognized.

Please refer the following URL to download SISTEMA software and its instruction manual.
IFA - Software-Assistant SISTEMA

Providing reliability data for safety of machinery

OMRON provides reliability data for safety of control components as a SISTEMA Library and Parameter list in order to assist customer's PL calculation.

Please download the library and confirm whether there is data corresponding to the type of device required.

About any other product not listed in the attached parameter lists, please inquire to Omron sales office in each region.
