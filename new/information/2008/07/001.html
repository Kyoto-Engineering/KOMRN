Our new Product Search function has been unveiled

July 25, 2008

Our new Product Search function has been unveiled.
It is now possible to search each item in more detail, and access desired information with just a click.
The new 'Suggest' function is convenient to use. Just input a partial product code.

How to access to and use the Product Search function

This function can be accessed at

  • http://www.ia.omron.com/product/index.html

1.The following improvements have been made

  • It is now possible to search for detailed specifications of each product.
  • A display of the classification of the retrieved information, and links to pertinent information have been added to search results.
  • The search function now suggests catalogue and item codes to aid your search, and provides links to catalogue and item pages.
  • Click on a product name to move to a page with detailed information about that product.
[Note]
  • "Catalogue" : e.g., E2E, E3Z
  • "Item" : e.g., E2E-C1B1 2M, E3Z-D61 2M

2.Display of the Number of Hits and Product information

When the partial code matches an item code

"Items" are displayed in search results:
The thumbnail and product names contain links to detailed pages.
There is a link to the related catalogue page at the bottom right.

When the partial code matches a catalogue code

"Catalogue" is displayed in search results:
The thumbnail and product names contain links to detailed pages.
There is a link to the Item Index page at the bottom right.

3.Regarding the suggest function

This function suggests catalogue and item codes to aid your search, and provides links to catalogue and item pages.
Click on a product name to move to a page with detailed information about that product.

On the Products Search pages

Inputting a partial code will display a list of matching catalogue and item codes.

Catalogue codes now contain links to catalogue pages

Even when a catalogue contains 0 items, a link to the catalogue page is provided.

Item codes now contain links to item pages

On the Item Index, Lineup pages

Product Search has been installed in the Item Index and Lineup pages.Inputting a partial item code will display a list of matching item codes.

Item codes now contain links to item pages

Only items related to the displayed catalogue are suggested as possible matches.
